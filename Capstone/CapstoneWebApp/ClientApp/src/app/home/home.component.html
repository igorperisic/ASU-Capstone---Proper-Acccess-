Event Reporter

Function:  sends an http request to receive the parsed JSON info taken from the UniFi Web Api.
You can select the specific event to display from the dropdown and it will display a table with data for that specific event.
The table displays the time of occurance and the message of the event that occured.
There will be an option view all the data returned by the web api for that specific event in the advanced area
